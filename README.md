## Usage
1. Clone the repository
```bash
git clone https://github.com/aziz-0110/kai.git 
cd kai 
```
2. Build and activated the virtual environment
```bash 
virtualenv venv
source venv/bin/activate
```
3. Install requitments
```shell
bash install.sh 
```
4. Run the app
```shell
python3 controler.py 
```
